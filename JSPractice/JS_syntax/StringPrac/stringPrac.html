String

String 객체의 유용한 property와 function을 알아보자

length

txt의 길이 반환

indexOf

문자열 내부의 특정문자열이 있는지 찾고 문자열이 시작되는 index를 반환하는 함수

문자열을 찾지 못하면 -1을 반환한다

lastIndexOf

문자열 내부에 특정 문자열이 존재하는지 찾는 것은 동일하나 중복된 문자열이 있으면

특정 문자열의 마지막에 발견된 문자열의 index를 반환하는 함수

특정 문자열을 찾지 못하면 -1을 반환

slice

silce 함수는 파라미터로 시작 위치와 종료 위치를 주면, 문자열에서 해당 부분은 잘라내서 반환

마지막 5번 인덱스는 포함하지 않는다

첫 번째 파라미터인 시작 위치 인덱스를 음수로 주면, 문자열의 끝에서부터 거꾸로 읽게 된다

substring

substring() 함수는 slice() 함수와 동일한 기능을 한다

단, 파라미터로 음수를 허용하지 않는다는 차이가 있습니다

첫 번째 인덱스,끝인덱스 파라미터로 넣기 끝 인덱스는 들어가지 않는다

두 번째 파라미터를생략하면, 시작 인덱스 위치에서 문자열의 마지막까지 반환한다

첫 번째 파라미터인 시작 위치 인덱스를 음수로 주면, 문자열의 끝에서 부터 거꾸로 읽게 된다

replace()

replace() 함수는 문자열 내의 특정 문자열을 지정한 문자열을 바꾸는 함수이다

이때 바꾸려는 문자열이 하나 이상 있더라도 처음 발견된 문자열만 바꾸게 됩니다

concat()

concat 함수는 2개 이상의 문자열을 하나의 문자열로 합치는 함수입니다

trim()

trim 함수는 문자열의 앞,뒤 공백을 제거하는 함수입니다

padStart와 padEnd 함수는 문자열 앞에 지정된 문자를 지정된 길이만큼 추가하는 함수

charAt() 함수는 문자열에서 특정 인덱스에 해당하는 문자 하나를 반환하는 함수입니다
